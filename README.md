## рҹ“қ Щ…Ш№ШұЩҒЫҢ

**ЩҫЩҲШұЩҒШ§ЫҢ** ЫҢЪ© ШіЫҢШіШӘЩ… Ш¬Ш§Щ…Ш№ ЩҒШұЩҲШҙ ЩҒШ§ЫҢЩ„ ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ШўЩҶЩ„Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ ШЁШ§ **Щ„Ш§ШұШ§ЩҲЩ„** ЩҲ **ЩҒЫҢЩ„Ш§Щ…ЩҶШӘ** ШӘЩҲШіШ№ЩҮ ЫҢШ§ЩҒШӘЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ ШіЫҢШіШӘЩ… ШЁШ§ Ш§ШұШ§ШҰЩҮ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Ъ©Ш§Щ…Щ„ Щ…ШҜЫҢШұЫҢШӘ Щ…ШӯШӘЩҲШ§ШҢ ЩҒШұЩҲШҙ Щ…ШӯШөЩҲЩ„Ш§ШӘ ШҜЫҢШ¬ЫҢШӘШ§Щ„ШҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ШўЩҶЩ„Ш§ЫҢЩҶ ЩҲ Щ…ШҜЫҢШұЫҢШӘ Ъ©Ш§ШұШЁШұШ§ЩҶШҢ ЫҢЪ© ЩҫЩ„ШӘЩҒШұЩ… ЩӮШҜШұШӘЩ…ЩҶШҜ ШЁШұШ§ЫҢ Ъ©ШіШЁвҖҢЩҲЪ©Ш§ШұЩҮШ§ЫҢ ШўЩҶЩ„Ш§ЫҢЩҶ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.

## вңЁ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ

### рҹ”§ ЩҶШөШЁ ЩҲ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ
- **ЩҶШөШЁвҖҢЪ©ЩҶЩҶШҜЩҮ Ш®ЩҲШҜЪ©Ш§Шұ** ШЁШ§ ШұШ§ШЁШ· Ъ©Ш§ШұШЁШұЫҢ ШІЫҢШЁШ§ (ШЁЩҲШӘвҖҢШ§ШіШӘШұЩҫ)
- ШӘЩҶШёЫҢЩ… Ш®ЩҲШҜЪ©Ш§Шұ ЩҒШ§ЫҢЩ„ `.env`
- Ш§ЫҢЩ…ЩҫЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ШҜЫҢШӘШ§ШЁЫҢШі
- Ш§ЫҢШ¬Ш§ШҜ APP_KEY ШЁЩҮ ШөЩҲШұШӘ ШӘШөШ§ШҜЩҒЫҢ

### рҹ’і ШҜШұЪҜШ§ЩҮ ЩҫШұШҜШ§Ш®ШӘ
- **Ш§ШӘШөШ§Щ„ ШЁЩҮ ШҜШұЪҜШ§ЩҮ ШІШұЫҢЩҶвҖҢЩҫШ§Щ„** (Zarinpal)
- Щ…ШҜЫҢШұЫҢШӘ ШӘШұШ§Ъ©ЩҶШҙвҖҢЩҮШ§ ЩҲ ЩҲШ¶Ш№ЫҢШӘ ЩҫШұШҜШ§Ш®ШӘ
- ШөШҜЩҲШұ ЩҒШ§Ъ©ШӘЩҲШұ Ш®ЩҲШҜЪ©Ш§Шұ

### рҹ“Ұ Щ…ШҜЫҢШұЫҢШӘ Щ…ШӯШөЩҲЩ„Ш§ШӘ
- Щ…ШӯШөЩҲЩ„Ш§ШӘ ШЁШ§ ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ
- ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩӮШ§ШЁЩ„ ШҜШ§ЩҶЩ„ЩҲШҜ ШЁШұШ§ЫҢ ЩҮШұ Щ…ШӯШөЩҲЩ„
- ЩҶЩ…Ш§ЫҢШҙ ЩӮЫҢЩ…ШӘ ЩҲ ШӘШ®ЩҒЫҢЩҒ
- ЩҲШ¶Ш№ЫҢШӘ Ш§ЩҶШӘШҙШ§Шұ (ЩҒШ№Ш§Щ„/ШәЫҢШұЩҒШ№Ш§Щ„)

### рҹ“қ Щ…ШҜЫҢШұЫҢШӘ Щ…ШӯШӘЩҲШ§
- **ЩҲШЁЩ„Ш§ЪҜ** ШЁШ§ ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ
- **Щ…ШҜЫҢШұЫҢШӘ ШөЩҒШӯШ§ШӘ** (ШҜШұШЁШ§ШұЩҮ Щ…Ш§ШҢ ШӘЩ…Ш§Ші ШЁШ§ Щ…Ш§ШҢ ...)
- **ШіЫҢШіШӘЩ… ШҜЫҢШҜЪҜШ§ЩҮвҖҢЩҮШ§** (ЩҶШёШұШ§ШӘ ЩҲ Ш§Щ…ШӘЫҢШ§ШІШ§ШӘ)
- ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШіШҰЩҲ (SEO)

### рҹ‘Ҙ Щ…ШҜЫҢШұЫҢШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ
- **ШіЫҢШіШӘЩ… Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘ** Ъ©Ш§Щ…Щ„ (ЩҲШұЩҲШҜ/Ш«ШЁШӘвҖҢЩҶШ§Щ…)
- **ЩҫЩҶЩ„ Ъ©Ш§ШұШЁШұЫҢ** Ш§Ш®ШӘШөШ§ШөЫҢ
- **ЩҫШұЩҲЩҒШ§ЫҢЩ„ Ъ©Ш§ШұШЁШұЫҢ** ШЁШ§ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҲЫҢШұШ§ЫҢШҙ
- **Щ…ШҜЫҢШұЫҢШӘ Щ…ШӯШөЩҲЩ„Ш§ШӘ Ш®ШұЫҢШҜШ§ШұЫҢ ШҙШҜЩҮ**
- **Щ…ШҜЫҢШұЫҢШӘ ЩҒШ§Ъ©ШӘЩҲШұЩҮШ§** ЩҲ ШӘШ§ШұЫҢШ®ЪҶЩҮ Ш®ШұЫҢШҜ

### рҹҺ« ШіЫҢШіШӘЩ… ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ
- **ШӘЫҢЪ©ШӘвҖҢЩҮШ§ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ** ШЁШ§ Ш§ЩҲЩ„ЩҲЫҢШӘвҖҢШЁЩҶШҜЫҢ
- ШЁШ®ШҙвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ (ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢШҢ ЩҒШұЩҲШҙШҢ Щ…ШҜЫҢШұЫҢШӘ)
- ЩҫШ§ШіШ®вҖҢШҜЩҮЫҢ ШҜЩҲШ·ШұЩҒЩҮ (Ъ©Ш§ШұШЁШұ/Ш§ШҜЩ…ЫҢЩҶ)
- ЩҲШ¶Ш№ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҫЫҢЪҜЫҢШұЫҢ (ШЁШ§ШІШҢ ЩҫШ§ШіШ® ШҜШ§ШҜЩҮШҢ ШЁШіШӘЩҮ)

### рҹ‘ЁвҖҚрҹ’ј Щ…ШҜЫҢШұЫҢШӘ ЩҫЩҶЩ„ Ш§ШҜЩ…ЫҢЩҶ
- **ШҜШ§ШҙШЁЩҲШұШҜ Щ…ШҜЫҢШұЫҢШӘЫҢ** Ъ©Ш§Щ…Щ„ ШЁШ§ ЩҒЫҢЩ„Ш§Щ…ЩҶШӘ
- **Щ…ШҜЫҢШұЫҢШӘ ЩҶЩӮШҙвҖҢЩҮШ§ ЩҲ ШҜШіШӘШұШіЫҢвҖҢЩҮШ§** (RBAC)
- Щ…ШҜЫҢШұЫҢШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҲ Щ…ШҜЫҢШұШ§ЩҶ ШіЫҢШіШӘЩ…
- ШўЩ…Ш§Шұ ЩҲ ЪҜШІШ§ШұШҙвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ
- ШіЩҒШ§ШұШҙЫҢвҖҢШіШ§ШІЫҢ Ъ©Ш§Щ…Щ„ ЩҫЩҶЩ„

- **Frontend:** Bootstrap 5, Livewire, Alpine.js
- **Database:** MySQL/MariaDB
- **PHP:** 8.2+

- Composer
- MySQL/MariaDB
- Node.js & NPM (ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮ)

## рҹҡҖ ЩҶШөШЁ ЩҲ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ

### ШұЩҲШҙ Ш§ЩҲЩ„ - ЩҶШөШЁ Ш®ЩҲШҜЪ©Ш§Шұ (ШіШ§ШҜЩҮ)

1. ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ ШұШ§ ШұЩҲЫҢ ЩҮШ§ШіШӘ Ш®ЩҲШҜ ШўЩҫЩ„ЩҲШҜ Ъ©ЩҶЫҢШҜ
2. ЩҒШ§ЫҢЩ„ `installer.php` ШұШ§ ШҜШұ Щ…ШұЩҲШұЪҜШұ ШЁШ§ШІ Ъ©ЩҶЫҢШҜ
3. Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҜЫҢШӘШ§ШЁЫҢШі ЩҲ ШіШ§ЫҢШӘ ШұШ§ ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ
4. ШұЩҲЫҢ **ШҙШұЩҲШ№ ЩҶШөШЁ** Ъ©Щ„ЫҢЪ© Ъ©ЩҶЫҢШҜ
5. ЩҫШі Ш§ШІ Ш§ШӘЩ…Ш§Щ…ШҢ ЩҫЩҲШҙЩҮ `install` ЩҲ `installer.php` ШұШ§ ШӯШ°ЩҒ Ъ©ЩҶЫҢШҜ

### ШұЩҲШҙ ШҜЩҲЩ… - ЩҶШөШЁ ШҜШіШӘЫҢ

1. Ъ©Щ„ЩҲЩҶ Ъ©ШұШҜЩҶ ЩҫШұЩҲЪҳЩҮ
git clone https://github.com/your-username/pourfi.git

2. ЩҶШөШЁ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§
composer install
npm install

3. Ъ©ЩҫЫҢ ЩҒШ§ЫҢЩ„ Щ…ШӯЫҢШ·
cp .env.example .env

4. ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜЫҢШӘШ§ШЁЫҢШі ШҜШұ ЩҒШ§ЫҢЩ„ .env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=pourfi
DB_USERNAME=root
DB_PASSWORD=

5. ШӘЩҲЩ„ЫҢШҜ Ъ©Щ„ЫҢШҜ
php artisan key:generate

6. Ш§Ш¬ШұШ§ЫҢ Щ…Ш§ЫҢЪҜШұЫҢШҙЩҶвҖҢЩҮШ§
php artisan migrate

7. ШӘЩҲЩ„ЫҢШҜ Щ„ЫҢЩҶЪ© Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ
php artisan storage:link

8. Ш§Ш¬ШұШ§ЫҢ ШіШұЩҲШұ
php artisan serve

## рҹ”җ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ЩҲШұЩҲШҜ ЩҫЫҢШҙвҖҢЩҒШұШ¶

| ЩҶЩӮШҙ | Ш§ЫҢЩ…ЫҢЩ„ | ШұЩ…ШІ Ш№ШЁЩҲШұ |
|------|-------|----------|
| **Щ…ШҜЫҢШұ Ш§ШөЩ„ЫҢ** | `admin@site.com` | `00000000` |
| **Щ…ШҜЫҢШұ ЩҒШұЩҲШҙ** | `sale@site.com` | `00000000` |

вҡ пёҸ **ШӘЩҲШ¬ЩҮ Ш§Щ…ЩҶЫҢШӘЫҢ:** ЩҫШі Ш§ШІ ЩҲШұЩҲШҜШҢ ШӯШӘЩ…Ш§ЩӢ ШұЩ…ШІ Ш№ШЁЩҲШұ Ш®ЩҲШҜ ШұШ§ ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.

## рҹ“Ғ ШіШ§Ш®ШӘШ§Шұ ЩҫШұЩҲЪҳЩҮ

pourfi/
в”ңв”Җв”Җ app/
в”Ӯ   в”ңв”Җв”Җ Filament/          # ЩҫЩҶЩ„ Ш§ШҜЩ…ЫҢЩҶ ЩҒЫҢЩ„Ш§Щ…ЩҶШӘ
в”Ӯ   в”ңв”Җв”Җ Http/              # Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ ЩҲ Щ…ЫҢШҜЩ„ЩҲШұЩҮШ§
в”Ӯ   в”ңв”Җв”Җ Models/            # Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШҜЫҢШӘШ§ШЁЫҢШі
в”Ӯ   в””в”Җв”Җ Providers/         # ШіШұЩҲЫҢШівҖҢЩҫШұЩҲШ§ЫҢШҜШұЩҮШ§
в”ңв”Җв”Җ database/
в”Ӯ   в”ңв”Җв”Җ migrations/        # Щ…Ш§ЫҢЪҜШұЫҢШҙЩҶвҖҢЩҮШ§
в”Ӯ   в””в”Җв”Җ seeders/           # ШҜЫҢШӘШ§ЩҮШ§ЫҢ Ш§ЩҲЩ„ЫҢЩҮ
в”ңв”Җв”Җ public/
в”Ӯ   в””в”Җв”Җ install/           # ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҶШөШЁ
в”ңв”Җв”Җ resources/
в”Ӯ   в””в”Җв”Җ views/             # ЩҲЫҢЩҲЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұЫҢ
в”ңв”Җв”Җ routes/
в”Ӯ   в”ңв”Җв”Җ web.php            # ШұЩҲШӘвҖҢЩҮШ§ЫҢ ШіШ§ЫҢШӘ
в”Ӯ   в””в”Җв”Җ filament/          # ШұЩҲШӘвҖҢЩҮШ§ЫҢ Ш§ШҜЩ…ЫҢЩҶ
в””в”Җв”Җ .env                   # ШӘЩҶШёЫҢЩ…Ш§ШӘ Щ…ШӯЫҢШ·ЫҢ

## рҹ“Ҡ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҫЩҶЩ„ Ш§ШҜЩ…ЫҢЩҶ

### ШҜШ§ШҙШЁЩҲШұШҜ Щ…ШҜЫҢШұЫҢШӘЫҢ
- ШўЩ…Ш§Шұ Ъ©Щ„ЫҢ ЩҒШұЩҲШҙ ЩҲ Ъ©Ш§ШұШЁШұШ§ЩҶ

## рҹҡҖ Installation

### Method 1 - Automatic Installation (Easy)

1. Upload project files to your hosting
2. Open `installer.php` in your browser
3. Enter database and site information
4. Click **Start Installation**
5. After completion, delete the `install` folder and `installer.php`

### Method 2 - Manual Installation

1. Clone the project
git clone https://github.com/your-username/pourfi.git

2. Install dependencies
composer install
npm install

3. Copy environment file
cp .env.example .env

4. Configure database in .env file
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=pourfi
DB_USERNAME=root
DB_PASSWORD=

5. Generate application key
php artisan key:generate

6. Run migrations
php artisan migrate

7. Create storage link
php artisan storage:link

8. Start the server
php artisan serve
